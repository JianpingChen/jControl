decentMouseEvent Class

Module: elements

Represents a mouse event consistent across browsers. This object is passed to mouse event handlers (sender, event) as the last parameter.

Attributes

altKey. Read only.

true if the alt key was down when the mouse event was fired.

button. Read only.

The button number that was pressed when the mouse event was fired. (0: left, 1: middle, 2: right)

clientX. Read only.

The X coordinate of the mouse pointer in local (DOM content) coordinates.

clientY. Read only.

The Y coordinate of the mouse pointer in local (DOM content) coordinates.

ctrlKey. Read only.

true if the control key was down when the mouse event was fired.

currentTarget. Read only.

The element whose event listeners triggered the event

metaKey. Read only.

true if the meta key was down when the mouse event was fired.

relatedTarget. Read only.

The secondary target for the event, if there is one.

screenX. Read only.

The X coordinate of the mouse pointer in global (screen) coordinates.

screenY. Read only.

The Y coordinate of the mouse pointer in global (screen) coordinates.

shiftKey. Read only.

true if the shift key was down when the mouse event was fired.

target. Read only.

The element that triggered the event
