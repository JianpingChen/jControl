restCallBase Class

Module: comm

Base class for RESTful API calls.

Constructor

restCallBase

(
  • option
)

Parameters:

  • option Object

    an object with the following properties:

    className {string} not used for logic but for debugging purpose to identify the object;

    relativePath {string} relative path to a serviceProxy urlRoot;

    callbackParam {object} optional, additional parameter to pass back in successCallback, redirectCallback, or failurCallback;

    successCallback {function} function to call after the call succeeded. This function will be called with (data, callbackParam) when the call succeeded. This function is called when the response status is 200 - 299.

    redirectCallback {function} function to call after the call succeeded but returned status is a redirect, 300 - 399. (jqXHR, textStatus, responseText, callbackParam) will be passed into this function.

    failureCallback {function} function to call after the call failed. (jqXHR, textStatus, responseText, callbackParam) will be passed into this function. This function is called when the response status is 400 and above.

Returns:

useJsonIo

()

Whether JSON input/output should be used. Derived class is expected to override this function if JSON is not used. Default is true.

When this function returns true, MIME type of the request will be set to "application/json" and Accept header of the request will include "application/json". Callback functions will be invoked after JSON.parse is called on responseText. If parsing failed, failureCallback will be invoked. If parsing succeeded, successCallback will be invoked with parsed data (instead of text data).

When this function returns false, Callback functions will be invoked with responseText as data.

Returns:

boolean
