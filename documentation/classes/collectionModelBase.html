collectionModelBase Class

Module: observe

A general purpose base class for collection of models to wrap around a raw data array. Derived classes may define calculated properties by overriding getPropertyValue function.

Constructor

collectionModelBase

(
  • arrayDto
  • modelFactory
)

Parameters:

  • arrayDto Array of object

    normally generated by JSON.parse when data transportation object is an array

  • modelFactory Object

    Used to provide customized factory to create child data model or specialized data model for array of objects. This object must implement all public functions defined in modelFactory class.

subscribeToCollectionChange

(
  • callback
)

Subscribe to collection changed event with a callback function. When a model is added or removed from this collection, or this collection is cleared, collection changed event will be raised.

Parameters:

  • callback Function

    this function will be called with (sender, eventArg) where sender is this object and eventArg is a structure. When a model is added, eventArg structure is { "change": 'added', "item": model, "index": index of the newly added model within the collection after it is added }. When a model is removed, eventArg structure is { "change": 'removed', "item": model, "index": index of the model before it is removed from the collection }. When this collection is cleared, eventArg structure is { "change": 'cleared', "item": this collection }.

Returns:

current object to support fluent style coding.

subscribeToModelChange

(
  • callback
)

Subscribe to model changed event. When any property in this model, or any of its contained model, or any model contain in its collection models, changed, callback function will be called.

Parameters:

  • callback Function

    callback function when model changed. When model is changed, this function will be invoked with (sender, eventArg). sender is this object and eventArg contains attribute "source" whose value contains the source object that has been changed. Other attributes in the eventArg provide more information about the change. For property value change, it also contains all information a property change event argument contains. For collection change, it also contains all information a collection change event argument contains.

Returns:

current object to support fluent style coding.
